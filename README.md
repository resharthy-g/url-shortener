# URL Shortener

Live Demo: https://url-shortener-nj32.onrender.com

A web application that converts long URLs into short links and tracks link usage through click analytics.

## Features

- Shorten long URLs
- Automatic short code generation
- Redirect users to the original URL
- Track link click counts
- View all stored links in a dashboard
- Delete links from the dashboard


## Tech Stack

### Backend
- Python
- Flask
  
### Frontend
- HTML5
- CSS3

### Database
- SQLite



## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in your browser:

```text
http://127.0.0.1:5000
```
